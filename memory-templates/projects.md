```

- **OKR link**: [Which KR this maps to, if any]
- **Summary**: [1-2 sentences on what this is]
- **Current state**: [What's happening now]
- **Blockers**: [Any blockers, or "None"]
- **Last updated**: (as of YYYY-MM-DD)
```

(Projects will be populated as you use chiefOS — run `/signal-scan` or `/risk-radar` to seed.)
