# Suppliers & Partners

Track external suppliers, partners, and vendors here. Populated by `/setup` from `config/domain.md` and enriched by `/supplier-watch`.

## Format

```
### [Supplier Name]
- **Type**: [e.g. API provider, vendor, platform partner]
- **Owner**: [Internal person responsible for this relationship]
- **Status**: 🟢 Healthy / 🟡 Watch / 🔴 Critical
- **Integration**: [How they connect to your systems]
- **Known issues**: [Any ongoing concerns]
- **Last updated**: (as of YYYY-MM-DD)
```

(Run `/supplier-watch` to populate and monitor supplier health.)
